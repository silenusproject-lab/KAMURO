# プライバシーポリシー / Privacy Policy

## 日本語

最終更新日：2025年11月14日

### 1. はじめに

KAMURO（以下「本アプリ」）は、花火の音から距離を計算するiOSアプリケーションです。本プライバシーポリシーは、本アプリがユーザーの情報をどのように取り扱うかを説明するものです。

### 2. 開発者情報

**アプリ名**: KAMURO
**開発者**: Tsuyoshi Nishigaki
**連絡先**: silenus.project@gmail.com

### 3. 収集する情報

#### 3.1 位置情報

本アプリは、以下の目的で位置情報を使用します：

- **使用目的**：
  - 花火の打上地点を地図上で選択・表示するため
  - 「現在地を使用」機能で、ユーザーの現在位置を打上地点として設定するため
  - 計算結果を地図上に表示するため

- **取得方法**：
  - iOS標準のCore Locationフレームワークを使用
  - 精度：100メートル（kCLLocationAccuracyHundredMeters）
  - 取得タイミング：アプリ使用中のみ

- **重要事項**：
  - 位置情報は**端末内でのみ使用**され、外部サーバーには一切送信されません
  - 位置情報は保存されず、アプリを閉じると破棄されます
  - ユーザーはいつでもiOSの設定から位置情報の利用を無効化できます

#### 3.2 入力データ

本アプリは、ユーザーが入力する以下のデータを処理します：

- 時間差（秒数）
- 気温（摂氏）

**重要事項**：
- これらのデータは端末内でのみ計算処理に使用されます
- データは保存されず、外部に送信されることもありません

#### 3.3 アプリ設定

本アプリは、以下の設定情報を端末内に保存します：

- 言語設定（日本語/英語）

**保存場所**：
- iOS標準のUserDefaultsを使用
- 端末内のみに保存され、外部には送信されません

### 4. 収集しない情報

本アプリは以下の情報を**収集しません**：

- 個人を特定できる情報（氏名、メールアドレス、電話番号など）
- 端末の識別情報（UDID、広告IDなど）
- 使用統計や分析データ
- クラッシュレポート
- 閲覧履歴や検索履歴

### 5. 第三者へのデータ提供

本アプリは、ユーザーのデータを第三者に提供、販売、共有することは**一切ありません**。

### 6. 第三者サービスの利用

#### 6.1 Apple MapKit

本アプリは、地図表示のためにApple MapKitを使用します。

- **使用目的**：地図の表示、地名検索
- **データ送信**：地図タイル取得のため、Appleのサーバーと通信します
- **プライバシー**：MapKitの利用は、Appleのプライバシーポリシーに準拠します
- **詳細**：[Apple プライバシーポリシー](https://www.apple.com/jp/privacy/)

#### 6.2 SNSシェア機能

本アプリには、以下のSNSへのシェア機能があります：

- X (Twitter)
- note
- LINE

**重要事項**：
- シェア機能は、ユーザーが明示的にボタンをタップした場合のみ動作します
- 各SNSアプリまたはWebサイトへの遷移のみを行います
- 本アプリから直接データを送信することはありません
- 各SNSのプライバシーポリシーが適用されます

#### 6.3 App Store評価機能

本アプリは、iOS標準のStoreKitフレームワークを使用してApp Storeでの評価をリクエストします。

- **使用目的**：アプリの評価をユーザーに依頼するため
- **データ送信**：Apple以外の第三者には送信されません
- **詳細**：Appleのプライバシーポリシーに準拠します

### 7. データの保存期間

- **位置情報**：保存されません（アプリ終了時に破棄）
- **入力データ**：保存されません（計算後に破棄）
- **アプリ設定**：ユーザーがアプリを削除するまで端末内に保存

### 8. データのセキュリティ

本アプリは、以下のセキュリティ対策を実施しています：

- 全てのデータ処理は端末内で完結
- 外部サーバーへのデータ送信なし（地図表示を除く）
- iOS標準のセキュリティ機能を使用

### 9. お子様のプライバシー

本アプリは、13歳未満のお子様を含む全年齢の方が利用できます。本アプリは、年齢に関わらず個人情報を収集しません。

### 10. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。変更があった場合は、本ページに掲載し、アプリのアップデート情報でお知らせします。

### 11. お問い合わせ

本プライバシーポリシーに関するご質問やご不明な点がございましたら、以下の連絡先までお問い合わせください。

**メールアドレス**: silenus.project@gmail.com

## English

Last Updated: November 14, 2025

### 1. Introduction

KAMURO (hereinafter "the App") is an iOS application that calculates distance from fireworks sound. This Privacy Policy explains how the App handles user information.

### 2. Developer Information

**App Name**: KAMURO
**Developer**: Tsuyoshi Nishigaki
**Contact**: silenus.project@gmail.com

### 3. Information We Collect

#### 3.1 Location Information

The App uses location information for the following purposes:

- **Purpose of Use**:
  - To select and display the fireworks launch point on the map
  - To set the user's current location as the launch point with the "Use Current Location" feature
  - To display calculation results on the map

- **Collection Method**:
  - Uses iOS standard Core Location framework
  - Accuracy: 100 meters (kCLLocationAccuracyHundredMeters)
  - Collection timing: Only while using the app

- **Important Notes**:
  - Location information is **used only within the device** and is never transmitted to external servers
  - Location information is not saved and is discarded when the app is closed
  - Users can disable location services at any time through iOS settings

#### 3.2 Input Data

The App processes the following user-input data:

- Time difference (seconds)
- Temperature (Celsius)

**Important Notes**:
- This data is used only for calculations within the device
- Data is not saved or transmitted externally

#### 3.3 App Settings

The App saves the following settings information within the device:

- Language setting (Japanese/English)

**Storage Location**:
- Uses iOS standard UserDefaults
- Saved only within the device and not transmitted externally

### 4. Information We Do Not Collect

The App **does not collect** the following information:

- Personally identifiable information (name, email address, phone number, etc.)
- Device identification information (UDID, advertising ID, etc.)
- Usage statistics or analytics data
- Crash reports
- Browsing or search history

### 5. Data Sharing with Third Parties

The App **does not** provide, sell, or share user data with third parties.

### 6. Use of Third-Party Services

#### 6.1 Apple MapKit

The App uses Apple MapKit for map display.

- **Purpose**: Map display, place search
- **Data Transmission**: Communicates with Apple servers to retrieve map tiles
- **Privacy**: MapKit usage complies with Apple's Privacy Policy
- **Details**: [Apple Privacy Policy](https://www.apple.com/privacy/)

#### 6.2 SNS Sharing Features

The App includes sharing features for the following social networks:

- X (Twitter)
- note
- LINE

**Important Notes**:
- Sharing features only work when users explicitly tap the button
- Only transitions to each SNS app or website
- The App does not directly transmit data
- Each SNS's privacy policy applies

#### 6.3 App Store Review Feature

The App uses the iOS standard StoreKit framework to request App Store reviews.

- **Purpose**: To request app reviews from users
- **Data Transmission**: Not transmitted to any third parties other than Apple
- **Details**: Complies with Apple's Privacy Policy

### 7. Data Retention Period

- **Location Information**: Not saved (discarded when app closes)
- **Input Data**: Not saved (discarded after calculation)
- **App Settings**: Saved within device until user deletes the app

### 8. Data Security

The App implements the following security measures:

- All data processing is completed within the device
- No data transmission to external servers (except map display)
- Uses iOS standard security features

### 9. Children's Privacy

The App is suitable for all ages, including children under 13. The App does not collect personal information regardless of age.

### 10. Changes to Privacy Policy

This Privacy Policy may be updated as necessary. Changes will be posted on this page and announced in app update information.

### 11. Contact Us

If you have any questions or concerns about this Privacy Policy, please contact us at:

**Email**: silenus.project@gmail.com
